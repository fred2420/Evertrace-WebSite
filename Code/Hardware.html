Hardware Overview

Design constraints

The overarching principle of making the first evertrace board was to make a development board. A board that could be easily used for testing and experimentation before spawning a family of smaller purpose built NB-IOT products.

The PCB is built around the Atmel Atmega328p 8 bit microcontroller. The same microcontroller used on the Arduino UNO. Arduino is open source with a huge community online which cuts down the learning curve. Schematics for the Arduino UNO can be found here.

All components are were possible 0805 package size, this should allow easy testing soldering and re-soldering of components. All free GPIOs from the microcontroller are available and 6 debug LEDs are included on the board. (Jumper pins all the LEDs to be connected and removed).

Micro USB

The Board is both programed and powered through a Micro USB port. No reprogramming of the Quectel firmware is possible - I have never needed to do this.

Coin Battery

The coin battery provides power for the real time clock chip. Coin battery is used instead of a capacitor to avoid having to deal with inrush current and to allow the real time clock to be reset.
